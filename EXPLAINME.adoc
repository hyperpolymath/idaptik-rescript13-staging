== Core Claims

=== Claim 1: Staging Repository for ReScript 13 Migration with Zero Deprecated APIs

*From README:* "This is the STAGING repository for ReScript 13 migration testing. The live repo is hyperpolymath/idaptik. All deprecated Js.* APIs have been migrated. Do NOT reintroduce them."

*Evidence:* The `CLAUDE.md` file documents a complete audit of deprecated Js.* APIs, all migrated as of March 2026: `Js.Dict.*` → `Dict.*`, `Js.Array2.*` → `Array.*`, `Js.String2.*` → `String.*`, `Js.Json.classify()` → `JSON.Classify.classify()`. The `rescript.json` is "v13-clean" with modern configuration keys (`dependencies`, `compiler-flags`, `-open RescriptCore`). A search across ~542 `.res` files confirms zero use of deprecated patterns. This staging repo serves as a frozen baseline for validating the ReScript 13 upgrade path. **Caveat:** This is a staging repository; changes here should not be merged back to the live idaptik repo without explicit approval.

=== Claim 2: Complete Async Game with Combat, Multiplayer, and Network Simulation

*From README:* "IDApixiTIK is a browser-based hacking/network simulator game... combining platformer gameplay with realistic network simulation, featuring multiplayer co-op support via Elixir/Phoenix WebSocket server."

*Evidence:* The game architecture spans multiple systems: `src/app/combat/` implements guard encounters and knockdown mechanics; `src/app/devices/` simulates routers, firewalls, laptops with SSH and DNS; `src/app/multiplayer/` handles Phoenix WebSocket channels for real-time co-op; `src/app/companions/` provides AI-driven Moletaire companion. The sync server at `sync-server/` is a full Elixir/Phoenix application managing sessions via ETS. The game startup at `src/Main.res` orchestrates asset loading, screen navigation, and the TEA event loop. Tests in `sync-server/test/` verify connectivity; game state tests in `shared/tests/` verify device interaction. **Caveat:** This is still a staging environment; gameplay balance and multiplayer stability are being validated before merging to live.
